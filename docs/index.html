Test: Calculating the average

Write a Scratch program to calculate the average of some numbers entered by the user. You do not know how many numbers the user will choose to enter.

The algorithm for the code is provided below

When the program starts:
	Set the total to 0
	Set the count to 0
	loop forever:
		ask the user to enter the next number
		if the user does not enter anything or enters 0:
			set the average to total/count
			say the average
			stop the program
		otherwise:
			add the answer to the total
			add 1 to the count
